#### map(func, iter) function
return type: map object(iteratior)

예제1
```
def multiplication(n):
    return n * n

numbers = (1, 2, 3)
result = map(multiplication, numbers)
print(list(result)) 
#[1, 4, 9]
```
예제2 - lambda function활용
```
numbers1 = [1, 2, 3]
numbers2 = [3, 2, 1]
result = map(lambda x, y: x * y, numbers1, numbers2)
print(list(result))
#[3, 4, 3]
```
